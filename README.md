## ğŸ“– Learning Journey

### Step 1: Presentation (15-20 minutes)
Teacher-led presentation introducing friction, oil, and the 8 critical areas.

### Step 2: Hands-On Activity (30-40 minutes)
Students use interactive web app alongside Halfords model engines to identify:
1. Camshaft Bearings
2. Valve Train
3. Piston Rings
4. Cylinder Walls
5. Connecting Rod Bearings
6. Main Bearings
7. Crankshaft Journals
8. Oil Pump Gears

### Step 3: Individual Assessment (10-15 minutes)
Students take the single-player quiz to test their knowledge.

### Step 4: Class Competition (15-20 minutes)
Optional multiplayer quiz for revision and engagement.
